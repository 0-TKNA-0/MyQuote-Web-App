A web application built using Ruby and Ruby on Rails that allows a user to add and view their favourite quotes.
The web app has a front end interface that allows a user to create an account and add as many quotes as they like to the application.
A user can view any quote that is made publicly available by the user who posted the quote.
A user can edit their own quotes after it has been submitted.
It features a backend database that uses SQLite3 to store quotes and login data.
